## Diagram Guidelines

When creating new diagrams:

1. **Use standard formats**: SVG (preferred), PNG, or Mermaid
2. **Keep diagrams simple**: Focus on one concept per diagram
3. **Label clearly**: Use consistent terminology from TGP-GLOSSARY.md
4. **Show message flow**: Indicate direction with arrows
5. **Include legend**: Explain symbols and notation
